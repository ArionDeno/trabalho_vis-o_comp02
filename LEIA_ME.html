Introdução:


    O trabalho em questão, visa demostrar a aplicação das transformações, em duas imagens ao mesmo tempo aplicando-se : Rotação,Translação, Escalonamento e Virar.

     Para essa demostração, foi utilizado duas imagens fornecidas pelo professor, de uma mão e um relógiode pulso, que serão aplicada as transformações, em tempo real, utilizando-se a poderosa API multi plataforma, para jogos e aplicações multimídia >SDL2 e para carregar imagens no formato PNG em complemento SDL2_image.     As transformações ocorrem em tempo real, pois a aplicação usa conceitos amplamente usados em desenvolvimento de jogos, como: framerate, gameloop, renderização por hardware e controle do framerate, posicionamento relativo.

Recursos Ultilizados

Foi Ultilizado para esse trabalho A linguagem C e as seguintes ferramentas:

  • Linguagem C
  • Compilador GCC - 8.1
  • SDL2 - 2.0.12
  • SDL2_image - 2.0.0
  • Cmake - 2.8



      • As teclas de atalho para a execução das operações são:



      Teclas direcionais movem as imagens
      Tecla R Rotaciona Horario
      Tecla E Rotaciona Anti - Horario
      Tecla H Acresenta Escala
      Tecla G Diminiu Escala
      Tecla ESC Finaliza a Execução
